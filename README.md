# Installation

## Font

```sh
brew tap homebrew/cask-fonts
brew install --cask font-computer-modern
```

## This CLI tool

Clone and run:

```sh
make
```

## Development

## Precommit

Install [`pre-commit` tool](https://pre-commit.com)

```sh
brew install pre-commit
```

## Imagemagick

Install [`imagemagick`](https://imagemagick.org)

```sh
brew install imagemagick
```
